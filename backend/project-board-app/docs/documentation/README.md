# 一、概述

> An `awesome` project.

### 1.简介

这里是软件工程学院（滴水湖国际软件学院）课题发布与成员招募系统在线用户手册文档。

### 2.系统功能介绍

本系统旨在为我校教师和同学们提供一个科研互动的便捷交流平台。教师通过本系统发布需要招募组员的项目信息，管理学生的应聘申请和面试预约申请；同学们通过本系统浏览科研项目信息，管理个人简历和提交应聘申请，同时也可以订阅自己感兴趣领域的项目信息。
希望通过这个平台，可以让教师的科研项目需求和学生的科研兴趣得到快速匹配，让同学们参与到各类课题的研究工作中，提升教学和科研的互动性与效率。

# 二、教师端

教师管理端可通过PC浏览器登录，系统集成了本校公共数据库统一认证登录，无须开通单独的账号。
1. 登录方式一
   
   进入学院院内OA系统（https://seioa.ecnu.edu.cn）， 点击‘课题发布及成员招募系统’即可进入。
   
![系统流程概述](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/portal-project.png)

2. 登录方式二
   
   直接输入地址进入系统：https://seioa.ecnu.edu.cn/project-board-ms
   

### 1.发布课题
   教师进入系统后，在`课题项目管理`模块可查看我发布的课题列表、添加新的课题、修改课题信息、添加课题岗位需求、修改课题发布状态和修改课题成员招募状态。
   
   教师进入管理后台后，界面如下：

![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/teacher-project-list.png)
   
   点击右上角`添加`按钮后，在弹窗界面录入课题信息。信息录入过程中，打星号字段为必填内容，其余字段可选择性填写。Tips，课题信息披露越详细，越容易找到合适的学生组员！
  
![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/teacher-project-new.png)
   
   添加课题后，还需进入课题详情添加课题的岗位需求。注意：如果未添加课题岗位需求，学生将无法提交应聘申请！课题岗位需求添加界面如下：
  
![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/teacher-project-role.png)
   
   <strong>注意：只有课题状态处于`在研`且成员招募状态处于`成员招募中`的课题才能让学生正常报名应聘！</strong>   

### 2.管理学生应聘申请
   学生提交课题申请后，将通过企业微信消息的方式推送给课题负责老师，然后老师可进入系统对学生申请进行审批。
   
   学生申请审批界面如下：
  
![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/teacher-appointment-approval.png)
   
### 3.管理学生面试预约
   学生应聘申请通过之后，方可进行面试预约。如果教师和学生已有线下沟通，也可省去面试预约的环节，直接将该学生的申请设置为`面试通过，确认录用`。

# 三、学生端
   学生目前仅开放了移动端访问入口，可通过企业微信或者微信进入系统。同样的，学生登录系统也集成了公共数据库统一认证登录，无须额外的账号。
1. 登录方式一

   进入华师大企业微信，查找`滴水湖国际软件学院`应用，通过应用菜单链接`课题平台`进入系统。
   
![系统流程概述](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/wework-entrance.png)

2. 登录方式二

   直接通过微信或者企业微信客户端进入系统：https://seioa.ecnu.edu.cn/project-board-client， 也可微信或者企业微信扫码进入。

![系统流程概述](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/client-qrcode.png)


### 1.浏览课题
学生进入系统H5客户端后，界面如下：

![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/client-project-list.png)

点击底部菜单`个人中心`进入查看个人简历、面试预约和我提交的课题应聘申请。学生个人简历维护界面如下：

![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/client-resume.png)

在课题列表页面浏览课题，可点击某个课题进入查看课题详情。通过课题详情可以查看课题相关研究方向和课题成员需求信息，通过Tab切换可以查看不同的岗位要求。界面如下：

![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/client-application-detail.png)

### 4.查看我的申请
课题申请提交后，进入`个人中心`查看我提交的申请，界面如下：

![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/client-application-list.jpg)

### 5.提交面试预约
在`个人中心`点击`面试预约`后，可以进入面试预约提交界面，界面如下：

![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/teacher-project-list.png)

一个课题应聘申请对应一个面试预约，界面如下：

![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/client-appiontment-detail.png)

### 6.管理我的课题订阅
学生可以设置课题状态订阅，在系统发布相关订阅方向的课题时，会主动向订阅的学生推送课题信息，界面如下：

![课题列表](https://seioa.ecnu.edu.cn/docs/project-board-manual/img/client-subscribe.png)


# Working On It !
本系统将极大提升教师课题发布、学生成员招募、团队协作的效率，同时为学生提供更多的实践机会，促进教师与学生的互动交流，推动科研教学的深入发展。

后续本系统将持续进行迭代优化，进一步完善个人简历数据、从信息办统一拉取学生个人数据、使用大模型提升智能化体验、UI和交互升级优化等。

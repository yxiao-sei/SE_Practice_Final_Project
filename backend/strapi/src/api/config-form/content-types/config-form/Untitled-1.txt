在此之际，人工智能（AI）以出人预料的方式迅速崛起。早在上世界八十年代中期，钱学森就指出，AI是人类的未来，AI属于工程技术层面，对其起支撑作用的应该是思维科学。在知识掌握上，AI能力将不断提升，人们已无法与之抗衡；人们只有在思维科学层面发力，才有能力保住人类的尊严。
人类的思维方式和思维体系林林总总，哪些思维能够作为当代AI时代人类思考的基石，能让人举一反三、触类旁通，去应对AI时代的挑战？量子思维和老子思维，可谓其中极能代表、极能趋近“第一性原理”的思维体系。
量子思维无疑起源于量子学说，但其已超越传统的微观物质科学层面。20世纪以来的两次量子革命极大推动了人类科技和社会发展，量子思维的测不准、叠加、跳跃等属性越来越显著地呈现出来，也愈加深刻地影响着科学技术的发展。
量子信息论在人文社会科学领域的成功应用，都让人们能够以新的视角环顾世界，回溯过去和面对未来。因为有了量子论才有了能带理论和半导体以及后来的计算机与互联网；毫无疑问人工智能也是以此基础发展而来，这一轮的崛起更是深諳其道而引发的变革。
老子等远古中国智者的思维及其对世界规律的认识，在古代成为中国科技发展和富强朝代社会治理的重要基石。汤川秀树等众多诺贝尔奖获得者和学术界智者崇敬老子思维，指出此思维的当代性和未来价值；玻尔甚至将太极图立为家徽并认为该思想就是对量子论最好的描绘。我们当代不缺乏对新技术、新结果、新发现的追捧和热议，而缺乏对深层次影响人类未来的“第一性思维”的探讨和畅想。
在上述背景下，我们诚邀全球在AI、量子、老子及其相互渗透领域的智者汇聚上海，召开“量子老子国际论坛”，以“人工智能之道”为主题，基于量子思维、老子思维和它们的相互融通，讨论三方面的问题：1）AI伦理与老子思想；2）AI未来与量子科技；3）AI与未来思维教育。此次论坛的主题"Tao of AI"（人工智能之道），旨在汇聚全球顶尖的研究者、打破传统学科的界限，推动全球学术界在人工智能与人类思维的交汇点上发现新的知识领域，提升华东师范大学在人工智能、量子老子研究领域的显示度和引领力。</p><p style="--tw-border-spacing-x: 0;  document.getElementById('toBottom').addEventListener('click', function(event) {
    event.preventDefault(); // 阻止链接默认行为

    // 获取固定头部元素的高度
    const headerHeight = document.getElementById('header').offsetHeight;

    // 目标滚动位置：页面底部减去固定头部的高度
    const targetScroll = document.documentElement.scrollHeight - window.innerHeight;

    // 设置滚动的总时间（2秒）
    const duration = 2000;

    // 获取当前滚动位置
    let start = 0;

    // 开始滚动动画的时间
    let startTime = null;

    // 缓动函数
    function easeInOutQuad(t) { 
        return t < 0.5 ? 2 * t * t : -1 + (4 - 2 * t) * t;
    }

    // 滚动动画
    function animateScroll(timestamp) {
        if (!startTime) startTime = timestamp;
        const elapsed = timestamp - startTime;
        
        // 计算当前时间与开始时间的比例
        const progress = elapsed / duration;
        
        // 使用缓动函数创建更平滑的运动效果
        const ease = easeInOutQuad(progress);
        
        // 更新滚动位置
        window.scrollTo(0, start + (targetScroll - start) * ease);
        
        // 请求下一帧动画，如果时间未到则继续动画
        if (elapsed < duration) {
            requestAnimationFrame(animateScroll);
        } else {
            // 确保最终位置正确
            window.scrollTo(0, targetScroll);
        }
    }
    
    // 开始滚动动画
    requestAnimationFrame(animateScroll);
});

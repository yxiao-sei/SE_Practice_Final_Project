# yjsyxt-web

这是一个使用 Vite 构建的 Vue 项目。

## 描述

`yjsyxt-web`是一个基于 Vue 3 的项目，使用了 Vite 作为构建工具。项目中使用了 Element Plus 作为 UI 框架，并使用了 Axios 进行 HTTP 请求。此外，项目还使用了 Echarts 进行数据可视化。

## 安装

以下是安装此项目的步骤：

1. 克隆仓库：`git clone https://github.com/your/repo.git`
2. 进入项目目录：`cd yjsyxt-web`
3. 安装依赖：`npm install`

## 运行

以下是运行此项目的步骤：

1. 开发模式：`npm run dev`
2. 构建项目：`npm run build`
3. 预览项目：`npm run preview`

## 代码检查

运行 ESLint 进行代码检查：`npm run lint`

## 构建

以下是不同环境下的构建命令：

1. 生产环境：`npm run build:prd`
2. UAT 环境：`npm run build:uat`
